## 3줄 요약

1. **시장은 실재하고 크다.** 미국·캐나다 히스패닉+한국계 owner-operator 계약자 약 32~43만 명, TAM USD $1.2~1.3억. 경쟁사 중 모국어 AI 견적 도구는 단 0개.
2. **매출 성장의 핵심 레버는 세 가지다.** 무료 티어를 월 3건으로 강화, 연간 결제 옵션 도입, 레퍼럴 프로그램 — 이 세 가지가 6~9개월 안에 $10K MRR을 만든다.
3. **제품 로드맵은 '신뢰' 중심으로 쌓아야 한다.** 오프라인 모드 → QuickBooks 연동 → 사진 견적 → 팀 협업 순서로 진행해야 이탈을 막고 업셀을 끌어낼 수 있다.

## 1.2 TAM / SAM / SOM

```
TAM: USD $1.2~1.3억/년 (미국 히스패닉+한국계+캐나다 히스패닉+한국계 계약자 전체)
SAM: USD $3,600만~6,000만/년 (스마트폰 보유 + SaaS 인지 가능 + 언어 조건 충족)
SOM: USD $720만~3,000만/년 (Year 3 목표: $1,500만)
```

**왜 SAM이 TAM의 30~40%인가:**
- 스마트폰 보급률 85~90%가 기술적 상한선
- SaaS 인지 및 디지털 결제 경험자 비율: ~50%
- 실제 견적 발행이 필요한 사업자: ~70%

## 1.3 1차 타겟: 히스패닉 계약자 (Beachhead)

### 왜 히스패닉이 1순위인가

| 근거 | 데이터 |
|------|--------|
| 절대 규모 | TAM의 90%+ 차지 |
| 성장 속도 | 전체 건설업의 2.5~3배 성장 |
| 통증 심각도 | 견적 포기율 30~40%, 영어 서류가 사업 장벽 1위 |
| 현존 솔루션 | 스페인어 AI 견적 도구: 0개 |
| 지역 집중도 | 텍사스·플로리다·캘리포니아 3개 주에 집중 |

### 집중 지역 (우선순위 순)

| 지역 | 히스패닉 계약자 집중도 | 추천 이유 |
|------|---------------------|----------|
| **텍사스** (댈러스·휴스턴·샌안토니오) | 최고 | 미국 최대 히스패닉 건설 시장 |
| **플로리다** (마이애미·올랜도) | 높음 | 빠른 성장 + 라틴계 커뮤니티 강함 |
| **캘리포니아** (LA·샌호세) | 높음 | 절대 인구 최대 |
| **캐나다 온타리오** | 중간 | 홈마켓 + CAD 가격 유리 |

### 집중 업종 (우선순위 순)

1. **배관 (Plumbing)** — 견적 빈도 높음, 긴급 견적 필요
2. **전기 (Electrical)** — 복잡한 자재 목록 → AI 가치 큼
3. **HVAC** — 계절성 있지만 단가 높음
4. **지붕 (Roofing)** — 히스패닉 비중 높음, 현장 즉시 견적 패턴
5. **리모델링/타일** — 한국계 집중 업종

## 2.3 경쟁사 심층 프로파일

### Jobber — 시장 리더, 영어권 SMB 중심

**강점:**
- 종합 FSM (Field Service Management) — 스케줄링, CRM, 인보이싱, 결제 통합
- 브랜드 인지도, 대규모 사용자 커뮤니티
- QuickBooks/Stripe 등 주요 통합

**약점:**
- 스페인어·한국어 지원 없음
- 온보딩 복잡도 높음 (15~30분)
- Core $39 USD도 비영어권 계약자에게 "비쌈"으로 인식
- 견적 생성: 수동 입력 방식

**SnapQuote 기회:** Jobber의 English-only 온보딩 장벽을 이용. "Jobber의 절반 가격, 모국어 지원" 포지셔닝.

### Housecall Pro — 플러스 기능 강조

**강점:**
- GPS 추적, 현장 자동배정, 카드 리더기
- Essentials부터 QuickBooks 통합

**약점:**
- Essentials $149 USD/월 — 소규모 계약자에게 진입 장벽
- 완전 영어 전용
- AI 기능 없음

**SnapQuote 기회:** Housecall Pro 대비 7~9배 저렴한 진입 가격. 1인 계약자에게 과도한 기능이 없는 린 버전.

### Contractor+ — 가장 직접적인 저가 경쟁자

**강점:**
- Free Forever 플랜 (5건/월)
- $29 USD/월로 저렴
- 모바일 중심

**약점:**
- 영어 전용
- AI 없음
- 음성 입력 없음

**SnapQuote 기회:** Contractor+의 기능 집합 + 스페인어/한국어 + AI 음성 = 명확한 업그레이드. CAD $34 vs USD $29은 유사 가격대이므로 기능 차별화가 핵심.

### Joist / Invoice2go — 초저가 단순 앱

**강점:** 무료 또는 매우 저렴, 단순 사용

**약점:** 견적 생성이 완전 수동, AI 없음, 다국어 없음

**SnapQuote 기회:** Joist 무료 사용자를 SnapQuote 무료로 전환 후 유료화 가능. "Joist에서 업그레이드" 광고 전략 유효.

# 3. 매출성장 전략

## 3.1 현행 수익 진단

**현재 문제:**
- Starter CAD $34 (~USD $25): Jobber Core($39 USD)의 64% → **저평가** 상태
- 무료 티어 10건/월: 솔로 계약자의 주 2~3건 패턴을 커버 → **전환 압박 없음**
- 연간 결제 없음: 이탈률 관리 레버 부재

**업계 Freemium 전환율 벤치마크:**
| 전환율 | 위치 |
|--------|------|
| 2~5% | SMB SaaS 중앙값 |
| 5~10% | 상위 25% |
| 8~12% | Best-in-class |
| **7~8%** | **SnapQuote Year 1 목표** |

## 3.2 무료 티어 재설계 (가장 높은 레버)

### 현행 vs 권장

| 기능 | 현행 | 권장 | 이유 |
|------|------|------|------|
| 월 견적 건수 | **10건** | **3건** | 주 1회 패턴 계약자도 빠르게 소진 |
| AI 언어 지원 | 전체 | **영어만** | 스페인어/한국어 = 유료 전환 핵심 동기 |
| PDF 워터마크 | 없음 | **"Powered by SnapQuote"** | 바이럴 루프 활성화 |
| SMS 발송 | 있음 | **유료만** | 핵심 유용성 기능 유료 전환 |
| 견적 이력 | 무제한 | **30일** | 데이터 접근 = 업그레이드 동기 |
| 클라이언트 포털 | 있음 | **유료만** | 고객 관계 = 전문성 기능 |

**기대 효과:** 전환율 +3~5%p (약 2~4주 내 효과 발현)

## 3.3 가격 구조 권장안 (CAD 기준)

### Phase 1 즉시 적용 (현재)

## 3.6 전환 최적화 전술 (우선순위 순)

### T1. 사용량 벽 메시지 (즉시 효과 최강)
2번째 견적 발송 시점에 인앱 메시지:
> "이달 무료 견적 2/3 사용. 스타터로 업그레이드하면 무제한 + SMS 발송 — CAD $37/월"

→ 기대 효과: 전환율 +3~5%p

### T2. 온보딩 활성화 시퀀스
- Day 0 이메일: "3분 안에 첫 견적 보내보세요"
- Day 3 이메일: "견적이 열렸어요 — 후속 SMS 보내는 법"
- Day 7 이메일: "이번 달 무료 건수 2건 남았어요 — 업그레이드하기 전에 써보세요"

→ 기대 효과: 90일 이내 이탈률 -15~20%

### T3. PQL (제품 자격 리드) 트리거
다음 조건 충족 시 24시간 내 개인화 메시지 발송:
- Week 1에 2건 이상 견적 발송
- 스페인어/한국어 기능 사용
- 프로 가격 페이지 2회 이상 조회

→ PQL 전환율 25~30% (일반 freemium 대비 5~6배)

### T4. PDF 워터마크 바이럴 루프
```
무료 견적서 → "Powered by SnapQuote" 문구
    ↓
고객/일반인이 견적서 받음 → 앱 인지
    ↓
2~5%가 SnapQuote 가입
    ↓
바이럴 계수(K): 0.1~0.3 (초기)
```

### T5. 업그레이드 시점 연간 결제 프롬프트
업그레이드 클릭 시 기본으로 연간 결제 옵션 표시:
> "대부분의 계약자가 연간 결제를 선택합니다 — CAD $82 절약"

→ 연 결제 비중: 20% → 35~40%로 증가

## 4.4 기능별 상세 분석

### P0: 오프라인 모드 — 가장 시급한 신뢰성 기능

**왜 P0인가:**
- 건설 현장 = 인터넷 불안정한 환경 (지하, 외벽, 창고)
- 현재 30초 약속이 현장에서 실패하면 사용자 이탈 즉시 발생
- 현장에서 고객 앞에서 앱이 멈추면 신뢰 타격 = 해지

**구현 접근:**
```
1. 견적 폼 로컬 저장 (IndexedDB / SQLite)
2. 음성 → 텍스트 변환만 온라인 필요 (캐싱 처리)
3. AI 생성은 오프라인 초안 저장 → 연결 시 자동 완성
4. "오프라인 모드" UI 인디케이터 표시
```

### P0: 레퍼럴 프로그램 — 가장 낮은 CAC 채널

**구현 명세:**
- 레퍼럴 링크 1탭 생성 (스페인어/한국어/영어 공유 문구 자동 생성)
- 보상: 레퍼러에게 1개월 무료, 신규 가입자에게 2주 Pro 트라이얼
- 추적: 레퍼럴 대시보드 (내가 초대한 사람 현황)
- 스페인어 템플릿: "Oye, esta app te hace cotizaciones en español en 30 segundos 👷"

### P1: 사진 견적 — Pro 업셀 핵심 기능

**가치 제안:** 현장에서 사진 찍으면 → AI가 자재 목록 + 면적 + 예상 비용 자동 계산

**왜 Pro에만:** 이 기능이 있어야 Starter에서 Pro로 업그레이드할 이유가 생김. 현재 Pro와 Starter의 차이가 불분명.

### P1: QuickBooks 연동 — 중형 계약자 락인 기능

**계약자 워크플로:**
```
SnapQuote에서 견적 생성 → 수주 → QuickBooks에서 인보이스 자동 생성 → 결제
```

현재 이 과정을 수동으로 두 번 하고 있는 계약자가 많음. 연동 후 이탈률 대폭 감소.
